# Day 14 – Mini JavaScript Project

## What I Built
- Task Manager application using vanilla JavaScript

## Features Implemented
- Add, edit, delete tasks
- Task completion toggle
- Filter tasks by status
- Persistent state using localStorage

## Concepts Applied
- DOM manipulation
- Event-driven programming
- Array methods (map, filter)
- State-driven UI rendering

## Key Learnings
- UI should be rendered from state, not manually updated
- Clean separation between logic and presentation
- Small projects combine many core JavaScript concepts
