# Day 16 – React Fundamentals (Conceptual)

## What I Learned
- Why React was created and what issues it helps solve  
- The basic idea behind components  
- What JSX is and why it’s useful

## Key Concepts
- Manually working with the DOM becomes difficult as the UI gets bigger
- React automatically keeps the UI updated when data changes  
- The UI is split into small, reusable pieces called components  
- JSX allows us to write UI in a more readable, declarative way using JavaScript  

## Important Takeaways
- React focuses on defining *what* the UI should look like instead of handling updates manually  
- Components make the codebase cleaner and easier to reuse  
- JSX is still JavaScript, just written in a UI-friendly syntax  

## Notes
- React uses a virtual DOM to manage updates efficiently  
- Data flows in one direction, from parent to child
