# Task Manager (React)

A production-ready Task Manager application built with React,
focused on clean architecture, predictable state management,
and real-world UX behavior.

## Features
- Add, edit, delete tasks
- Inline editing
- Filters (All / Active / Completed)
- Visual feedback for completed tasks
- Smooth micro-animations
- Persistent state using localStorage
- Clean component-based architecture

## Tech Stack
- React
- JavaScript
- CSS

## Key Concepts
- useState for state management
- useEffect for side effects
- Controlled inputs
- Derived UI state
- Component composition

## Why this project?
This project demonstrates how a well-structured Vanilla JavaScript
application can be cleanly translated into React when fundamentals
are strong.
