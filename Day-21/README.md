# Day 21 – Conditional Rendering & Lists in React

## What I Learned
- How React conditionally renders UI
- How to render lists using map
- Why keys are required in lists

## Conditional Rendering Patterns
- && for showing UI conditionally
- Ternary operator for if–else UI
- Early return for clean logic

## Rendering Lists
- Use map() to convert data to UI
- Each list item must have a key

## Key Concept
- Key helps React track list items between renders
- Keys must be unique and stable

## Notes
- Avoid using array index as key when list changes
- Keep JSX readable when using conditions
